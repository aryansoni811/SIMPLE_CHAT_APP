# SIMPLE_CHAT_APP
simple chat app build using chatgpt

Working emojis:
'react'"⚛"
'hey' "👋"
'like' "❤"
'lol' "😂"
'woah'"😮"

anything that starts with '/' will not get sent to all only to client side. 
Special Commands: 
/help 
/ranodm 
/clear
